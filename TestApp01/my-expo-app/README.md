# My Expo App

This is a simple React Native application built using Expo and file-based routing via Expo Router.

## Getting Started

To get started with this project, follow the instructions below.

### Prerequisites

Make sure you have the following installed:

- Node.js (version 14 or later)
- Expo CLI

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/my-expo-app.git
   ```

2. Navigate to the project directory:

   ```bash
   cd my-expo-app
   ```

3. Install the dependencies:

   ```bash
   npm install
   ```

### Running the App

To run the application, use the following command:

```bash
npm start
```

This will start the Expo development server. You can then open the app on your mobile device using the Expo Go app or in an emulator.

### File Structure

- `app/index.tsx`: Main entry point of the application.
- `app/about.tsx`: About screen component.
- `app/_layout.tsx`: Layout component for managing navigation.
- `assets/icon.png`: Application icon.
- `package.json`: NPM configuration file.
- `app.json`: Expo app configuration.
- `tsconfig.json`: TypeScript configuration file.
